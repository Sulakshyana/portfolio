A modern developer portfolio built with **Next.js 16 (App Router)** showcasing my projects, professional experience, technical skills, and blogs.  
Designed with a strong focus on **performance, SEO, scalability, and clean architecture**.

## About the Project

This portfolio serves as a central place to present my work as a **Full Stack / Frontend-focused Developer**.  
It includes:

- Personal and professional overview
- Highlighted projects
- Technical skillset
- Blog system powered by MongoDB
- SEO and PWA best practices using Next.js built-in features

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript, JavaScript (ES6)
- **Frontend:** React 19, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express.js, REST APIs
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT
- **Tools:** Git, GitHub, Postman, VS Code
